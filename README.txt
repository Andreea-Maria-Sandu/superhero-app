# Superhero App

Web application for superhero management built with FastAPI and MySQL.

## Features
- Authentication (JWT)
- Role-based access (admin / user)
- Superhero listing with pagination
- Search & filtering
- Admin CRUD operations (Insert / Update / Delete)

## Tech Stack
- Backend: FastAPI, SQLAlchemy
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Auth: JWT

## How to run
1. Create virtual environment
2. Install dependencies
3. Configure database
4. Run backend with uvicorn
5. Open frontend in browser
